# Mutual DECAM - Bienestar Integral

Este proyecto es una plataforma web moderna para la **Mutual DECAM** (Mutual Deportes de Contacto, Artes Marciales y Afines), enfocada en brindar servicios de bienestar, salud y solidaridad.

## ✨ Características principales
- **Diseño Moderno:** Implementación de estilos limpios con una paleta cálida y profesional.
- **Interactividad:** Animaciones fluidas utilizando `Framer Motion`.
- **Experiencia de Usuario (UX):** Carga optimizada de imágenes mediante técnicas de *Skeletons* y *Blur-up*.
- **Gestión de Servicios:** Filtrado dinámico de actividades (Reiki, Deportes, Acción Social, etc.).

## 🛠️ Tecnologías utilizadas
- [React](https://reactjs.org/) - Librería de JavaScript para la interfaz.
- [Vite](https://vitejs.dev/) - Herramienta de construcción rápida.
- [Framer Motion](https://www.framer.com/motion/) - Biblioteca para animaciones.
- [CSS3](https://developer.mozilla.org/es/docs/Web/CSS) - Estilos personalizados (sin frameworks externos para mayor control).
